## Hi there 👋

I build browser-facing tools for **3D visualization**, **geospatial scenes**, and **large 3D data workflows**.

My current lane is practical 3D infrastructure: converting messy data, organizing it into streamable formats, inspecting spatial alignment, and making heavy scenes feel smoother in the browser with **Three.js**, **CesiumJS**, **3D Tiles**, and **Gaussian Splatting**.
